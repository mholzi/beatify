Installation

Requires: Home Assistant 2024.1+ · Music Assistant, Sonos, or Alexa

📦 HACS Recommended

One-click install via the Home Assistant Community Store.

  1. Open HACS in Home Assistant
  2. Click the button below
  3. Follow the prompts & restart

🔧 Manual Install

For advanced users.

  1. Open HACS → Custom Repositories
  2. Add mholzi/beatify as an Integration
  3. Install Beatify → Restart Home Assistant
  4. Go to Settings → Devices & Services → Add Integration → Beatify
